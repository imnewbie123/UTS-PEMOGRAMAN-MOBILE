# 🍽️ Masakan Nusantara App

Aplikasi Android sederhana berbasis **Jetpack Compose** untuk menampilkan informasi restoran, menu makanan, dan fitur edit profil.

---

## 📱 Fitur Aplikasi

✅ Menampilkan profil restoran  
✅ Edit profil (nama, alamat, deskripsi, jam buka)  
✅ Menampilkan daftar menu makanan  
✅ Navigasi antar halaman (Home, Menu, Detail, Profile)  
✅ Detail menu dengan interaksi jumlah pesanan  

---

## 🛠️ Teknologi yang Digunakan

- Kotlin
- Jetpack Compose
- Navigation Compose
- SharedPreferences
- Material 3 Design

---

## 📂 Struktur Halaman

- **HomeScreen** → Halaman utama + info restoran  
- **MenuScreen** → Daftar menu makanan  
- **DetailMenuScreen** → Detail menu + jumlah pesanan  
- **EditProfileScreen** → Edit data restoran
